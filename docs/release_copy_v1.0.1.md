# Sazoo 출시 문구/릴리스 노트 (v1.0.1)

## 1) 스토어 한줄 소개 (짧은 설명)
AI 사주 분석과 3D 힐링 공간이 만난, 나만의 운세 상담 앱 Sazoo.

## 2) 스토어 상세 설명 (국문)
오늘의 운세를 텍스트로만 보던 시대는 끝났습니다.  
Sazoo는 전통 사주 해석에 AI와 3D 인터랙션을 결합해, 더 몰입감 있고 개인화된 운세 경험을 제공합니다.

주요 기능
- 정밀 사주 온보딩: 이름, 성별, 생년월일, 출생시간 기반 분석
- AI 사주 마스터 채팅: 내 사주 컨텍스트를 이해하는 맞춤형 상담
- 3D 홈 공간: 한옥/천문 테마의 감성적인 인터랙티브 씬
- 오행 에너지 밸런스: 목/화/토/금/수 흐름 시각화
- 대운 주기/오늘의 운세: 일상에서 바로 확인 가능한 핵심 리딩
- 미니앱: 포춘쿠키, 부적 등 가볍게 즐기는 운세 콘텐츠

이런 분들께 추천해요
- 매일 나를 위한 운세 인사이트를 받고 싶은 분
- 단순 결과가 아닌 대화형 해석이 필요한 분
- 감성적인 UI/3D 경험을 좋아하는 분

안내
- 사주/운세 콘텐츠는 자기이해를 돕는 참고 정보입니다.

## 3) 스토어 상세 설명 (영문)
Sazoo combines traditional Saju (Korean fortune reading), AI guidance, and immersive 3D scenes for a deeply personalized daily insight experience.

Key Features
- Precise onboarding with name, gender, birth date, and birth time
- Context-aware AI Saju Master chat
- Interactive 3D sanctuary scenes
- Five-element energy balance visualization
- Daily fortune and luck-cycle highlights
- Mini apps like fortune cookie and amulet cards

Note
- Fortune content is provided for reflection and entertainment.

## 4) ストア詳細説明 (日本語)
Sazooは、四柱推命ベースの運勢解釈にAIガイドと3Dインタラクションを組み合わせた、没入型の占いアプリです。

主な機能
- 名前/性別/生年月日/出生時間によるパーソナライズ分析
- 文脈を理解するAI「Saju Master」チャット
- 3Dホーム空間でのインタラクティブ体験
- 五行バランスの可視化
- 今日の運勢と運気サイクルの表示
- フォーチュンクッキー/お守りなどのミニアプリ

注意
- 占いコンテンツは自己理解の参考情報として提供されます。
